----------------------------------------------------------
WICED Application Snippet Directory - README
----------------------------------------------------------

how to use various WICED libraries and API functions.

A matrix indicating which features are demonstrated in each of
the snippet applications is included below for convenience.

Please read the <WICED-SDK>/Apps/README.txt *BEFORE* trying any
of these applications!
